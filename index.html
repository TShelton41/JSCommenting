JS Doc Commenting

This will show you how to comment JS Code.

Great tutorial site. https://www.valentinog.com/blog/jsdoc/. The NPM reps is located here. https://www.npmjs.com/package/jsdoc

Install JSDocs Locally

npm install --save-dev jsdoc

Install JSDocs Globally

npm -g install jsdoc

Run JSDocs if installed Globally

This is the command if you are running locally. This also changes the destination folder to docs. It will also grab two different js files and compile them. 

jsdoc -d docs app/app.js app/table.js

Run JSDocs if installed Locally

This is the command if you are running locally. This also changes the destination folder to docs. It will also grab two different js files and compile them. 

node_modules/jsdoc/jsdoc.js -d docs app/app.js app/table.js
